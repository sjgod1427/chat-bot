FEASIBILITY AND VIABILITY

The proposed solution is both feasible and viable as we have the required
technologies and skillset to implement the same and it can be scaled later on.

Using existing AI and NLP frameworks to create a chatbot and developing a
recommender system using TensorFlow is possible.
A Flutter application can be easily scaled for a large number of users using the
right data-storage platforms and API calls.
A basic marketplace with a payment page can be easily implemented and
scaled as and when the need arises.
Partnerships with local authorities and government aid can make it a viable
long-term solution.

The potential challenges to the solution can be as follows:

Collection of legal and reliable data for model training.

Curating high-quality traditional handicrafts and potential scams.

Convincing users to switch to our app from other conventional platforms.

These challenges can mostly be addressed with government aid since:

Access to official records will no longer be a challenge.

Seller authentication can be done on a legal basis.

Advertisement and promotion will be a lot easier.
